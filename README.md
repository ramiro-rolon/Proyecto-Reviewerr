# Reviewerr

## Overview

**Reviewerr** is a desktop application developed as a group academic project during the second year of the **Systems Analysis** degree.

The objective of the project was to design and build a platform where users can **review, rate, and explore movies and TV series**, 
while administrators can analyze user interactions and moderate platform activity.

The project combines application development with database design, focusing on structured data management and user interaction.

---

## Technologies Used

- **C#** – Main programming language used to develop the desktop application  
- **MySQL** – Database management system used for storing and managing data  
- **MySQL Workbench** – Used for database modeling and structure design  
- **Visual Studio** – Development environment for the application  
- **Miro** – Used for project planning and system diagramming  

---

## Application Features

The application allows users to create a personal profile where they can interact with different movie and TV show records.

### User Features

- Create and manage personal profiles  
- Rate and review movies and TV series  
- Access detailed information about films and series  
- View information such as:
  - Actors and cast members
  - Genre classification
  - Available streaming platforms  

### Administrator Features

Administrator accounts have additional permissions that allow them to:

- Analyze data generated by user interactions
- Moderate inappropriate language
- Manage platform activity
- Ban users when necessary

---

## Database Design (My Role)

My primary responsibility in the project was the **design, development, and maintenance of the entire database structure**.

My work included:

- Designing the relational database schema
- Defining the core entities and attributes
- Creating relationships between tables
- Ensuring data consistency and logical structure
- Supporting the application with a scalable and organized database model

The database was designed using **MySQL Workbench** and integrated with the application through **C# and Visual Studio**.

# Reviewerr

## Descripción

**Reviewerr** es una aplicación de escritorio desarrollada como un proyecto académico grupal durante el segundo año de la carrera de **Análisis de Sistemas**.

El objetivo del proyecto fue diseñar y desarrollar una plataforma donde los usuarios puedan **opinar, calificar y explorar películas y series**, 
mientras que los administradores pueden analizar las interacciones de los usuarios y moderar la actividad dentro de la plataforma.

El proyecto combina desarrollo de software con diseño de bases de datos, enfocándose en la correcta estructuración y gestión de la información.

---

## Tecnologías utilizadas

- **C#** – Lenguaje principal utilizado para desarrollar la aplicación  
- **MySQL** – Sistema de gestión de bases de datos utilizado para almacenar y organizar la información  
- **MySQL Workbench** – Herramienta utilizada para el modelado y diseño de la base de datos  
- **Visual Studio** – Entorno de desarrollo utilizado para programar la aplicación  
- **Miro** – Utilizado para la diagramación y planificación del proyecto  

---

## Funcionalidades de la aplicación

La aplicación permite que los usuarios creen un perfil personal desde el cual pueden interactuar con distintos registros de películas y series.

### Funcionalidades para usuarios

- Crear y administrar perfiles personales  
- Calificar y opinar sobre películas y series  
- Acceder a información detallada sobre distintos contenidos  
- Consultar datos como:
  - Actores y protagonistas
  - Género al que pertenece cada film o serie
  - Plataforma de streaming donde se encuentra disponible  

### Funcionalidades para administradores

Los perfiles de administrador cuentan con permisos adicionales que les permiten:

- Analizar información generada a partir de las interacciones de los usuarios
- Moderar el uso de lenguaje inapropiado
- Gestionar la actividad dentro de la plataforma
- Banear usuarios en caso de ser necesario

---

## Diseño de la base de datos (Mi rol en el proyecto)

Mi responsabilidad principal dentro del proyecto fue el **diseño, desarrollo y mantenimiento de toda la estructura de la base de datos**.

Entre mis tareas se incluyeron:

- Diseñar el esquema relacional de la base de datos  
- Definir las entidades y atributos principales  
- Crear las relaciones entre tablas  
- Garantizar la consistencia y correcta organización de la información  
- Integrar la base de datos con la aplicación desarrollada en C#

La base de datos fue diseñada utilizando **MySQL Workbench** y posteriormente integrada con la aplicación a través de **Visual Studio**.
